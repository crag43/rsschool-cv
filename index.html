Number of People in the Bus kata from Codewars:

There is a bus moving in the city, and it takes and drops some people at each bus stop. You are provided with a list (or array) of integer pairs. Elements of each pair represent the number of people getting on (first item) and off (second item) the bus. Your task is to return the number of people who are still on the bus after the last bus station. Even though it is the last bus stop, the bus is not empty, and some people are probably still there. 
var number = function(busStops){
  sum = 0;
  for (let i = 0; i < busStops.length; i++){
    for (let j = 0; j < busStops[i].length; j++){
      if (j == 0){
        sum += busStops[i][j];
      } else {
        sum -= busStops[i][j];
      }
    }
  }
  return sum;
}
